Этот скрипт — меню-утилита для управления SSH-туннельными пользователями и прокидывание порта 127.0.0.1 с Linux-сервера на Локальную машину 127.0.0.1. Позволяет:

Создавать пользователей с оболочкой /sbin/nologin (только SSH-туннели, без интерактивного входа)
Удалять пользователей из системы и из списка
Менять пароль (вручную или сгенерировать случайный)
Показывать список созданных пользователей с их статусом
При создании выводит готовую команду для проброса порта через SSH. Данные о пользователях хранит в /etc/tunnel_users/users.list.
